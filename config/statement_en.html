The Goal

Be the last player with a marker still on the board

Tsuro is simple turn based board game where you have to be the last man standing.
Tsuro is played by placing tiles on the board and creating paths that your marker will follow. Just be careful though, if the path you or another player makes sends you off the board you have lost.

  Rules

Tsuro is played on 6x6 board. During the game players are placing square tiles with different paths drawn on them on the board. Tiles are stored in tile deck, that consists of 35 tiles, until players draw the tiles from it.

First turn
The active player places his or her marker on one of the start marks on the outside edge of the game board.

Game turn
Game turn consists of 3 phases:
  1. Drawing tiles
  2. Placing single tile
  3. Move the markers

Drawing tiles
Tiles are drawn from the shuffled deck. Current player can have at most 3 tiles in his hands. If there are no tiles in deck remaining, player can have less then 3 tiles in his hand. This move is automated.

Placing single tile
During this phase player have at least 1 tile in his hand. He can place single tile from his hand on the board. The tile can be placed only at the end of player's current path. The tiles has id assigned and can be rotated clockwise 0, 1, 2 or 3 times and placed on the board. Your output should consist only of 2 numbers tileId and tileRotation

Move the markers
The active player moves his marker to the open end of the path.
All other players next to the new tile are moved to the ends of their new paths.
If any player leaves the board will be eliminated. If more then one player is eliminated during resolution phase, active player is scored worse then inactive ones.

  Game Input

Input - First turn

Line 1: opponents - count of opponents

Next opponents lines: 3 space separated integers representing opponent position - column row index or -1 -1 -1 if opponent is yet to start his/her move.

Output - First turn
A single line containing 3 space separated integers representing desired starting position - column row index
Input - Other turns

opponents lines:

  • tileId - id of last tile opponent played
  • tileRotation - orientation of last tile opponent played
  • col - current opponent position - column
  • row - current opponent position - row
  • index - current opponent position - index

tileCount - tiles on hand

tileCount lines:

  • tileId
  • four pairs of ps - index where path starts
    pe - index where path ends

Output - Other turns
A single line containing tileId and tileRotation
Constraints
... contains between 1 and 500 characters.

Allotted response time to output is
  • ≤ 1 second in first move
  • ≤ 50 ms in other moves
