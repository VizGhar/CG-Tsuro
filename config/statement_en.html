The Goal

Be the last player with a marker still on the board

  Rules

Tsuro is played on 6x6 board. During the game players are placing square tiles on the board with different paths drawn on them. After the move, players are following the new paths created. Tiles are stored in tile deck, that consists of 35 tiles.

First turn
The active player places his or her marker on one of the start marks on the outside edge of the game board. Each tile has 2 path ends on each side, so there are 2 x 6 x 4 = 48 possible starting position. Starting position should be selected by calling START column row index

Game turn
Game turn consists of 3 phases:
  1. Placing single tile
  2. Move the markers
  3. Drawing tiles

Placing single tile
During this phase player have at least 1 tile in his hand. He can place single tile from his hand on the board. The tile can be placed only at the end of player's current path. The tiles has id assigned and can be rotated clockwise 0, 1, 2 or 3 times and placed on the board. You can place the tile by calling PLACE tileId tileRotation

Move the markers
The active player moves his marker to the open end of the path.
All other players next to the new tile are moved to the ends of their new paths.
Any player that leaves the board will be eliminated. If more then one player is eliminated during resolution phase, active player is scored worse then inactive ones.

Drawing tiles
Tiles are drawn from the shuffled deck. Current player can have at most 3 tiles in his hands. If there are no tiles in deck remaining, player can have less then 3 tiles in his hand. This move is automated.

  Game Input

Input - Init

Line 1: opponents - count of opponents

Input - Game Loop

opponents lines:

  • tileId - id of last tile opponent played
  • tileRotation - orientation of last tile opponent played
  • col - current opponent position - column
  • row - current opponent position - row
  • index - current opponent position - index

tileCount - tiles on hand

tileCount lines:

  • tileId
  • four pairs of ps - index where path starts
    pe - index where path ends

Output - Game Loop
First turn START column row positionIndex Other turns PLACE tileId tileRotation
Constraints
Allotted response time to output is
  • ≤ 1 second in first move
  • ≤ 50 ms in other moves
